Use Cases

Each begins with one sentence and ends with a single confirmation.

Change wallpaper

Start

The user says, “I want to change my wallpaper.”

Preview

The agent displays several options with clear labels—such as “calm blue leaves”—so the choice is obvious.

Confirm

One action applies the new wallpaper.

Check storage / memory usage

Start

The user writes, “How much storage or memory is used?”

Summary

A clear card appears with simple bars and suggestions if space is tight.

Confirm

One action accepts the cleanup or closes the card.

Find nearby restaurants and book/queue

Start

The user says, “Find nearby restaurants.”

Unified list

The agent returns results from connected services with quiet and step‑free badges.

Confirm

A single action books a table or joins a queue without switching apps.

Hospital registration

Start

The user types, “Register at the hospital for next week.”

Slots

Available dates and required details are presented in plain language.

Confirm

One action submits and returns a ticket or queue number.

Coupons and lower prices

Start

The user says, “Show cheaper coupons online.”

Best option

The agent checks connected services and presents the best applicable coupon with a short explanation.

Confirm

One action applies it during booking or checkout.
